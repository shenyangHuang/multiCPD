README
The codebase and jupyter notebooks included here
can be used to replicate the real-world experiments
on the NYC TLC datasets.

The two variables that need to be updated in the notebooks are:
* NYC-MultiCPD-2016-Parameterized.ipynb: file_paths (in the Parameter block)
* NYC-MultiCPD-2020-Parameterized.ipynb: TLC_data_dir_path (in the Parameter block)
